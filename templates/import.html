Import from Excel / CSV

Export your Excel file as a CSV, make sure it has the right columns, then upload it below. All historical data will be added to the database.

  1. In Excel, go to File → Save As → CSV UTF-8 (.csv)
  2. Make sure your sheet has these 4 columns (add them if missing)
  3. Upload the CSV file below
ColumnRequired?Format / Example
dateYes2026-05-01 or 01/05/2026
typeYesincome or expense
amountYes5000 or 1,200.50
noteNoSupplier name, description, etc.
